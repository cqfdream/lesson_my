## es6 let const
    - let 用来声明变量
    - const 用来声明常量 必须赋初值

    - const arr = [1,2,3]
        - 在头部和尾部
        - unshift + shift
        - push pop
        常量能变吗
        - 简单数据类型 常量值不能改
        - 复杂数据类型 值可以改，但是类型不可以改
    - es6 之前没有常量
        var  常量的意义
        var  PI (用大写声明常量)
    - var  let  有什么区别?
        - let 支持块级作用域  let在块级作用域里就只能在块级作用域起作用，在全局作用域则可以在块级起作用，在使用之前定义即可
        - var 不支持块级作用域  它没有作用域的限制
